## Grüezi & Willkommen

Herzlich willkommen auf dem neuen Informatikgarten! Ich habe über die Weihnachtsferien die Seite auf einen komplett neuen Technologie-Stack umgestellt. Damit kommen viele Verbesserungen - und weitere werden folgen.

## Was ist neu?

### Der Code-Editor ist um einiges besser geworden. 
- Sie können die **Turtle mit der Maus herumziehen** und vergrössern.
- Sie können den **Editor als Ganzes vergrössern** oder in Vollbild nutzen.
- Ihr **Code wird zwischengespeichert**. Normalerweise lokal auf Ihrem Computer, aber wenn Sie sich mit Ihrem Microsoft-Account der Schule anmelden (rechts oben), wird der Code auch auf dem Server gespeichert.

```turtle
import turtle
eva = turtle.Turtle()

eva.forward(10)
```

### Mit Schul-Account anmelden
Sie können sich mit Ihrem Schul-Account anmelden, dann werden Ihre Codes auf dem Server gespeichert. Zukünftig wird uns das ermöglichen, Aufgaben und Code einfach zu teilen.

### Besserer Technologie-Stack
Das merken Sie (noch) nicht, aber die Webseite basiert jetzt auf der heutigen Standardtechnologie für moderne Webapps (nodejs, react, nextjs). Das heisst: Wir können den Informatikgarten viel leichter erweitern! Wenn Sie Ideen und wünsche haben, kommen Sie einfach auf mich zu.
